Put the two font files here (download from Google Fonts):

  Modak-Regular.ttf   -> used for the BLIPO title logo
                         https://fonts.google.com/specimen/Modak
  Play-Regular.ttf    -> used for all UI text (HUD, menus, buttons)
                         https://fonts.google.com/specimen/Play

Exact filenames matter; Board.gd loads these paths:
  res://fonts/Modak-Regular.ttf
  res://fonts/Play-Regular.ttf

If a file is missing the game still runs and falls back to Godot's default font.
